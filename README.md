# Fit Track 

FitTrack is a lightweight web application that helps users track their daily calorie intake and calories burned through workouts. The application dynamically calculates net calories and provides feedback on whether the user is in a calorie surplus, deficit, or balanced state.

This project is built using vanilla HTML, CSS, and JavaScript and focuses on practicing DOM manipulation, dynamic data rendering, and real-time calculations.

## Features
- Add food items with calories consumed
- Add workouts with calories burned
- Automatic calculation of:
-- Total calories consumed
-- Total calories burned
-- Net calories
- Dynamic table updates without page refresh
- Delete/edit entries
- Automatic clearing of input fields
- Status indicator for calorie balance
- Simple and responsive UI design
- Mobile-first responsive design

## Dashboard
- Displays a summary of:
-- Total calories consumed
-- Total calories burned
-- Net calorie balance
-- Status message

## Food Entry
- Allows users to add:
-- Food name
-- Calories consumed

## Workout Entry
- Allows users to add:
-- Exercise name
-- Duration
-- Calories burned

## Progress Feedback
Displays a dynamic message depending on the calorie balance.

## Tech Stack
- HTML5 – Page structure
- CSS3 – Layout and styling
- JavaScript (Vanilla) – Logic and DOM manipulation
